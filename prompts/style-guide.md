# PitchDreams Anime Style Guide

## Base Style Directive
Use this as the foundation for ALL character image prompts.

```
High-quality anime art in the style of Blue Lock and Captain Tsubasa.
Clean cel-shaded illustration, vibrant colors, detailed eyes with catchlight reflections.
Upper body portrait, slight 3/4 angle view.
Dark background with dramatic neon accent lighting.
Sports anime aesthetic with intensity and emotion.
Sharp linework, professional manga illustration quality.
No text, no watermark, no signature.
```

## Color Palette (Blue Lock Inspired)
- Background: Deep navy-black (#020814)
- Primary neon: Cyan (#00d4ff)
- Accent neon: Hot pink (#ff2d78)
- Energy: Electric blue (#4d7cff)
- Highlight: Neon green (#39ff14)

## Key Visual Elements
- **Eyes**: Large, detailed anime eyes with visible iris patterns and bright catchlights
- **Hair**: Dynamic, stylized — gravity-defying spikes, flowing strands with highlight streaks
- **Expression**: Intense and emotional — matches character personality
- **Lighting**: Dramatic backlighting with neon rim light on hair and shoulders
- **Background**: Dark with character-specific energy aura / speed lines
- **Clothing**: Soccer jersey with visible number, collar detail, realistic fabric folds

## Quality Keywords
Add these to ensure high-quality output:
- `masterpiece, best quality, highly detailed`
- `anime key visual, professional illustration`
- `dynamic lighting, cinematic composition`

## Negative Prompt (things to avoid)
```
low quality, blurry, deformed, extra limbs, bad anatomy,
text, watermark, signature, frame, border,
realistic photo, 3D render, western cartoon style,
chibi, super deformed, excessive fan service
```

## Image Specifications
- **Aspect Ratio**: Portrait 5:7 (matches 400x560 game viewport)
- **Minimum Resolution**: 1024px on shortest side
- **Background**: Keep simple/dark for easy game integration
- **Format**: PNG with transparency preferred (will be converted to webp)

## Gender Representation
- Both male and female characters should have equally dynamic, powerful poses
- Female characters: Athletic build, confident posture, same intensity as male characters
- Avoid stereotypical passive poses — all characters are competitive athletes

## Age Range Guidelines
- **Youth (10-14)**: Softer features, rounder eyes, slightly shorter proportions
- **Teen (15-18)**: Sharper features emerging, more angular, taller proportions
- **Adult (Coach/Parent)**: Mature features, defined jawline, broader build
